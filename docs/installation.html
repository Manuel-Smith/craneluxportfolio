Smoothwall installation in Oracle VirtualBox

April 13, 2023 Linux | Installation Guide

Smoothwall is an open source Linux based Firewall designed to protect networks from malicious traffic and unauthorized access. Some features shipped with Smoothwall include;

  1. Outbound and inbound web content filtering.
  2. Support for LAN, DMZ, and wireless networks.
  3. Timed access.
  4. Virtual Private Network.
  5. Portforwardinding, etc.

In this guide, we discuss how to install Smoothwall Express in Oracle VirtualBox.

Downloading the ISO file

One may download the ISO from SourceForge. Click here to download. For this guide, select the iso file Express-3.1-x86064.iso from the provided list.

Setup In VirtualBox

In your VirtualBox application:

  1. Click New. A dialog box will appear.
  2. Enter a Name to identify the Smoothwall Virtual machine.
  3. Select a Folder to store smoothwall configuration settings.
  4. Select the smoothwall ISO Image from the download directory.
  5. Click Next.
  6. Allocate a Base Memory of at least 2048 MB and 2 CPU cores.
  7. Click Next.
  8. Select the default Virtual Hard disk setting as Create a Virtual Hard Disk Now.
  9. Click Next.
  10. Confirm the Summary of your settings.
  11. Click Finish.

VirtualBox Network Configurations

Smoothwall will only work if at least 2 Network adapters are in Bridged mode.

  1. With the smoothwall instance selected, click the settings button.
  2. Click the Network tab.
  3. For Adapter 1, Enable the Network Adapter by clicking the check box.
  4. Select the adapter as a Bridged Adapter.
  5. Repeat this setting for Adapter 2.
  6. Click OK.

Installation

With the smoothwall instance selected:

  1. Click the Start button to start the installation. A new window instance will appear.
  2. Select the Boot Option, Install Smoothwall Express.
  3. Press OK for the next 3 dialog windows.
  4. Virtualbox will partition your hard drive afterward. Click OK after getting a congratulations message.
  5. Smoothwall will ask if you want to restore the configuration from a previous installation. Select No.
  6. Select Keyboard mappings as ua-ws [qwerty].
  7. Select the Timezone you are in from the provided list.
  8. Enter a hostname for your machine and press OK.
  9. Select the Default security policy as Open and press OK.

Network Configurations

Network configuration type

  1. Change the Network configuration type to Green + Red.

Card assignments

In this section, we assign network cards to the Green and Red Interfaces.

  1. Start with Green and press OK.
  2. Then Red and press OK. Youâ€™ll get a message. All cards successfully allocated.
  3. Press OK.

Address settings

In this section, we add IP addresses to the Green and Red Interfaces.

  1. Select Green.
  2. Press OK on the dialog box with a warning shows up.
  3. Enter the IP address for the Green Interface and its Network Mask.
  4. Press OK to proceed.
  5. Change the address settings for the RED Interface by selecting DHCP.
  6. Press OK to proceed.
  7. In the Address settings section, Press Done.

DNS and Gateway settings

In this section, we configure DNS and Gateway settings for the firewall.

  1. Add the Ip address for the primary DNS in IPV4 format.
  2. Add the IP address for the secondary DNS in ipv4 format.
  3. Enter the IP address of your router as the Default Gateway.
  4. Within the Network configuration menu section, press Done.
  5. Within the selection menu section, press Finished.

Web Admin and root password

Web admin password

  1. Enter a new admin password for the web interface. The password to use when logging into web admin pages.
  2. Confirm the password by entering it in the Again field.
  3. Press OK.

Root

  1. Enter a new root user password. One to use for the command line access.
  2. Confirm the password by entering it in the Again field.
  3. Press OK.

Congratulations! The setup is complete.

Press OK to reboot.

After rebooting, access the firewall instance in the web browser using IP address https://<<ip.addr>>:81, For example, http://192.168.88.2:81

The default Username is admin, and the password is one you set.
